Introduction
------------

MyResidence is a powerful tool to manage your towns and residence on
your Minecraft server with Bukkit server.

For installation and usage help, see:
http://www.github.com/hohl/MyResidence

Requirements
------------

- Java 6+
- Recent version of Bukkit
- iConomy, Essentials Conomy or BOSEconomy
- Permissions (2.X, 3.X) or GroupManager
