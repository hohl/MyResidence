Introduction
------------

MyResidence is a powerful tool to manage your towns and residences
on your Minecraft server with Bukkit.

For installation and usage help, see:
http://www.github.com/hohl/MyResidence

Requirements
------------

- Java 6+
- Recent version of Bukkit
- WorldEdit 4.X
- iConomy, Essentials Economy or BOSEconomy
